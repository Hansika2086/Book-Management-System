# Book Management System

A full-stack web application designed to manage book inventory efficiently. This is a web application built to help you keep track book collections. This system also includes CRUD operations, search functionality, and pagination. If you have a collection of books and want a simple way to manage them without using a messy spreadsheet, this app provides a clean, digital dashboard to do it.

## Features
* **Inventory Management:** Add, update, and remove book entries from the database.
Keep your library organized: You can add new books to your system, update their details if something changes, or remove them when you've finished reading or donated them.

* **Search & Filter:** Find books by title, author, or category.
Find books instantly: Instead of scrolling through a long list, you can use the search and filter features to quickly find a specific book.

* **Status Tracking:** Real-time visibility.
See what's happening: The app gives you a clear view of your collection so you know exactly what you have on your shelves at any time.

* **Pagination:** 
Seamlessly navigate through large collections.


## Tech Stack

* **Backend:** Java, Spring Boot, Spring Data JPA
* **Frontend:** HTML5, Tailwind CSS, JavaScript (Fetch API)
* **Database:** Mysql
