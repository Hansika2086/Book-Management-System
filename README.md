# Book Management System

A full-stack web application designed to manage book inventory efficiently. This system provides a clean interface for tracking book collections, including CRUD operations, search functionality, and pagination.

## Features

* **Inventory Management:** Add, update, and remove book entries from the database.
* **Search & Filter:** Instantly find books by title, author, or category.
* **Pagination:** Seamlessly navigate through large collections.
* **Status Tracking:** Real-time visibility into book availability.

## Tech Stack

* **Backend:** Java, Spring Boot, Spring Data JPA
* **Frontend:** HTML5, Tailwind CSS, JavaScript (Fetch API)
* **Database:** Mysql
